# Arduino Archive
## Project Overview
This repository is a technical archive of my robotics and embedded systems projects developed in high school. It demonstrates a clear evolution from basic electronic logic to integrated systems using **Arduino Uno/Nano** and **ESP8266**. This collection represents the foundation of my hands-on experience with hardware-software synchronization.

## Tech Stack
-   **Microcontrollers:** Arduino Uno, Arduino Nano, ESP8266 (Wi-Fi enabled).
-   **Environment:** Arduino IDE.
-   **Languages:** C++ (Arduino Sketch).

## Hardware & Integrated Components
I have extensive experience integrating a diverse range of peripherals into functional, real-world systems:
### Sensing & Data Acquisition
--   ***Environmental:*** Gas, Flame, Temperature, Humidity (DHT), and Light (LDR) sensors.
--   ***Distance & Motion:*** Ultrasonic (HC-SR04), Infrared (IR) sensors, and Joysticks.
### Actuators & Motion Control
--  ***Motors:*** DC Motors (with L298N drivers), Servo Motors, and Fan Motors.
--   ***User Interface:*** Potentiometers, Buttons, and Switches.
### Communication & Identification
--   ***Wireless:*** ESP8266 (Wi-Fi) and Bluetooth modules.
--   ***Security:*** RFID-RC522 module for access control and identification systems.
### Output & Visualization
--   ***Visual:*** 16x2 LCD Displays, RGB LEDs, UV LEDs, and standard LED arrays.
--   ***Audio:*** Speakers and Buzzers for alert and feedback systems.

## Technical Growth
-   **System Integration:** Learned to bridge the gap between physical sensor data and logical software execution.
-   **Troubleshooting:** Developed strong debugging skills for both hardware circuits and embedded C++ code.
-   **Foundational Logic:** These projects provided the necessary background for my more complex current work, such as the **ESP8266 Doom Port**.
