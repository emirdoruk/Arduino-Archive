# Arduino-Archive
The Arduino Projects I Have Made in High School

📌 Project Overview
This repository is a technical archive of my high school robotics and embedded systems projects. It demonstrates a transition from basic electronic logic to integrated systems using Arduino Uno/Nano and ESP8266.

🔌 Hardware & Integrated Components
I have experience integrating a wide variety of peripherals into functional systems:
Sensing & Data Acquisition:
Environmental: Gas, Flame, Temperature, Humidity, and Light (LDR) sensors.
Distance & Motion: Ultrasonic (HC-SR04) and Infrared sensors.
Actuators & Motion Control:
Motors: DC Motors (with L298N drivers), Servo Motors, and Fan Motors.
User Interface: Joysticks, Potentiometers, Buttons, and Switches.
Communication & Identification:
Wireless: ESP8266 (Wi-Fi) and Bluetooth modules.
Security: RFID-RC522 module for access control systems.
Output & Visualization:
Visual: 16x2 LCD Displays, RGB LEDs, UV LEDs, and standard LED arrays.
Audio: Speakers and Buzzers for alert systems.
